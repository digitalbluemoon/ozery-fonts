# Ozery Webfonts

To be used on Ozery Bakery projects.

Website's main font files: http://ozerybakery.com/Content/fonts.css

Client's font files: https://drive.google.com/open?id=0B-3QDDu6aQoYbWR2R2xLQmFxUjA
